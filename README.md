# Manipulação e Análise de Dados com Pandas

Este repositório dedica-se à apresentar os notebooks desenvolvidos durante 
o curso [Manipulação e Análise de Dados com Pandas - Python](https://www.udemy.com/course/manipulacao-e-analise-de-dados-com-pandas-python/) realizado na Udemy Online Courses pelo 
professor Rafael F. V. C. Santos, Ph.D., Cientista de dados e Gestor Estratégico de Riscos.

### Tecnologias

* Python 3
* Pandas
* NumPy
* Jupyter Notebook
* Matplotlib

### Conceitos trabalhados

* DataFrames e manipulação de dados
* Junção e agrupamento de DataFrames (Merge e Group By)
* Indexações de DataFrames
* Reshaping de dados
* Filtros booleanos
* Limpeza de dados
* Sumarização, duplicatas e ordenação
* Importação e exportação de dados (CSV, Excel, etc)
* Visualização de dados
* Tipos de gráficos
* Conceitos da EDA - Exploratory Data Analysis ou Análise Exploratória de Dados
